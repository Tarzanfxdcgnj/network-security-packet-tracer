# Network Security Packet Tracer Project

This project contains a simulated network built using **Cisco Packet Tracer** for the CM1103 – Computing Systems and Networking module.

## Overview

The aim of this lab was to design, configure, and test a small network, applying fundamental networking concepts and security practices. The simulation includes multiple devices such as routers, switches, and computers, demonstrating connectivity, IP addressing, and basic network troubleshooting.

## Environment Used

- Cisco Packet Tracer
- Simulated network devices: routers, switches, PCs
- Ubuntu/Linux environment (for reference)

## Key Tasks Performed

1. Designed a network topology with multiple devices.
2. Configured IP addressing and subnetting for each device.
3. Tested connectivity between devices using `ping` commands.
4. Implemented basic network security practices.
5. Captured screenshots of network layout and configuration for documentation.

## Skills Practiced

- Network topology design
- IP addressing and subnetting
- Device configuration (routers, switches)
- Connectivity testing and troubleshooting
- Basic understanding of network security principles
